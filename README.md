# Databricks Healthcare Data Analytics Project

## Project Title

**Health Vulnerability and Support Need Indicators in Canada**

## Project Overview

This project uses the 2022 Canadian Community Health Survey dataset to analyze population-level health vulnerability indicators in Canada. The goal is to identify groups that may show higher signs of health-related challenges based on self-reported health, mental health, chronic conditions, functional difficulty, healthcare access, and demographic or lifestyle factors.

This project is being built in Databricks as part of my hands-on learning in data analytics, big data processing, and healthcare analytics.

## Key Business Question

Which population groups in Canada show higher signs of health vulnerability and possible support need indicators based on the 2022 CCHS data?

## Dataset

Source: Canadian Community Health Survey 2022 Public Use Microdata File

The raw dataset is large and will not be uploaded directly to GitHub. The project will use selected columns and processed outputs for analysis.

## Planned Analysis

1. Load the CCHS 2022 CSV file into Databricks.
2. Review column names and dataset structure.
3. Select relevant columns for health vulnerability analysis.
4. Clean and transform coded survey values.
5. Analyze patterns by age group, province, income, chronic conditions, healthcare access, and functional difficulty.
6. Create summary tables and visualizations.
7. Prepare final insights for portfolio presentation.
